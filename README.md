🎓 Scholar Bio-Sync: Academic Performance Predictor
Optimizing Cognitive Load for High-Achieving Students
🎯 The Vision
Maintaining a 9.28 SGPI while balancing a Data Science internship and a rigorous daily commute is a significant data optimization problem. I built Scholar Bio-Sync to transform personal biometric data into actionable academic insights. This tool helps students determine their "Peak Focus" windows by analyzing the relationship between restorative sleep, environmental stressors, and cognitive workload.

🚀 The "Out of the Box" Features
Commute-Fatigue Integration: Specifically factors in the physical and mental drain of long-distance travel (like the Kalyan-Nerul route) to adjust performance expectations.

Dynamic Performance Gauge: Features a real-time visualization that shifts to Onion Pink (indicating low energy) or Growth Green (indicating peak focus) based on predictive metrics.

Impact Breakdown: Uses a probabilistic weighting system to show exactly which factor (Sleep vs. Stress) is currently dominating the user's cognitive capacity.

🧠 Technical Deep Dive
Predictive Engine: Employs a weighted regression logic to calculate focus scores, simulating how different variables interact in a real-world academic environment.

Interactive Framework: Built using IPyWidgets (for the Colab version) and Streamlit (for the web version), showcasing the ability to create functional, user-facing analytical tools.

Data Storytelling: Transforms raw numbers into visual narratives using Matplotlib and Pandas, making complex personal analytics easy to interpret at a glance.

🛠️ Tech Stack
Python: Core logic and data manipulation.

Scikit-Learn: (Optional) Used for training the Random Forest models for focus prediction.

Matplotlib & Seaborn: For high-fidelity biometric impact visualizations.

IPyWidgets: To facilitate a seamless interactive experience within research environments like Google Colab.
